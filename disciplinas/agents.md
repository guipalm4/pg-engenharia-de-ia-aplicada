# Agentes (ponteiro)

A referência oficial para agentes neste repositório fica em `.cursor/AGENTS.md`.

## O que o agente deve fazer
- Gerar/atualizar `README.md` por projeto seguindo `.cursor/templates/README_TEMPLATE.md`.
- Criar commits padronizados seguindo `.cursor/templates/COMMIT_TEMPLATE.md`.
- Usar `gh` (GitHub CLI) para operações com GitHub (auth/PR) quando aplicável.

## Fonte única de verdade
- Agents: `.cursor/AGENTS.md`
- Rules: `.cursor/rules/`
- Skills: `.cursor/skills/`
- Templates: `.cursor/templates/`
